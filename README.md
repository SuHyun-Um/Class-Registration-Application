## 📌 대학교 수강신청 APP서비스 소개
###  👩‍👩‍👧‍👦 APP 기능을 제공 합니다. 
> - 웹 뿐만 아니라 App 에서도 수강 신청을 할 수 있습니다.
> - 공지사항을 통해서 학교에 대한 정보를 빠르게 얻을 수 있습니다. 
### ✔️ 수강 신청 기능 제공
> - 수강 신청  및 수강 취소 기능 을 제공 합니다
> - 총점 계산 기능 기능을 제공 합니다
### 📈  강의 인기순위 분석 제공 
> - 전체 , 우리과 , 남자 선호도 , 여자 선호도 , 전공 인기도, 교양 인기도를 확인 할수 있습니다.
### 📆 달력을 통한 수강 신청 확인
> - 달력을 통해서 내가 신청한 강의 목록을 볼수 있습니다.

## 📌 기술 스택
> 해당 프로젝트를 수행하며 사용한 기술 스택입니다.

- Android
- Java 8 
- PHP
- MySQL
- 웹 호스팅 사용 (Cafe24)
- Gradle
